# AnimeSeek - Anime Scene Search & Download

A modern web application that allows users to search for anime scenes by uploading images and download anime series.

## Features

- **Image Search**: Upload anime screenshots to find the source anime, episode, and timestamp using trace.moe API
- **URL Search**: Search by providing an image URL
- **Anime Download**: Search and download anime series or individual episodes
- **Modern UI**: Beautiful, responsive design that works on all devices
- **Drag & Drop**: Easy image upload with drag and drop support

## Tech Stack

- Pure HTML5, CSS3, and JavaScript
- trace.moe API for anime scene recognition
- Jikan API (MyAnimeList) for anime information
- Font Awesome icons
- No build tools required
